📋 模板匹配规则设计

每条模板包含：触发条件（数据指标阈值）、段子文本、评级（S/A/B/C）、适用标签。一个玩家可匹配多条，按评级取最高分展示。

🎯 触发条件：常用英雄胜率 < 45% 且 场次 > 100
"你的 {champion} 玩了 {games} 场，胜率 {winrate}——我建议你去玩人机找找自信。"
英雄梗低胜率S级
🎯 触发条件：KDA 中死亡数 > 击杀数 * 1.5
"KDA {kills}/{deaths}/{assists}，你的死亡次数比你的补刀还稳定，对面打野看见你就像看见提款机。"
KDA梗高死亡率A级
🎯 触发条件：段位 < 黄金 且 总场次 > 500
"{tier} 段位，{total_games} 场——质量和数量你选了后者。峡谷质检员非你莫属。"
段位梗老玩家S级
🎯 触发条件：常用英雄为亚索/劫/永恩 且 胜率 < 48%
"又一个 {champion} 绝活哥——绝活指的是把队友活活气绝。E进人群的速度比回城还快，队友的问号信号都追不上你。"
英雄梗快乐风男S级
🎯 触发条件：胜率 > 55% 且 KDA > 3.0
"胜率 {winrate}，KDA {kda}——你是代练还是炸鱼？建议自首。不过说实话，玩得不错。"
高胜率夸奖B级

🤖 DeepSeek AI 调用方案

当模板未命中或匹配分不够时，构建提示词通过 DeepSeek 网页版对话接口生成个性化锐评。

// 构造锐评 Prompt const prompt = ` 你是一个LOL锐评机器人，说话风格毒舌、幽默、一针见血。 请根据以下玩家数据，生成一段100字以内的锐评： - 召唤师：{name} - 段位：{tier} - 胜率：{winrate} - KDA：{kda} - 常用英雄：{champions} - 总场次：{games} 要求： 1. 语气要像朋友间互相损，不要太恶意 2. 针对具体数据吐槽，不要泛泛而谈 3. 加点LOL玩家才懂的梗 4. 禁止人身攻击和脏话 `; // 通过 DeepSeek 网页版对话接口发送 sendToDeepSeek(prompt);

调用方式：模拟 DeepSeek 网页端 API（chat.deepseek.com 的对话接口），前端 fetch 请求，不需要 API Key。

🛡️ 安全 & 质量控制

敏感词过滤

  • • 脏话/人身攻击拦截
  • • 政治/敏感话题过滤
  • • 替换为温馨提示

质量兜底

  • • AI 超时 5s 回退模板
  • • 模板库至少 50+ 条
  • • 无数据时显示缺省文案
